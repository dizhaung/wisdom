Welcome to Wisdom Stream Processor

Wisdom is an adaptive, distributed and self-boosting stream processor written in Java 9 using modern architecture. After publishing all the results, Wisdom will be made open source.

If you like to use Wisdom for your research, please contact the author via: lgobinat@uwo.ca.

Requirements

  • Java 9 (or latest)
  • Apache Maven
  • Apache Kafka (for self-boosting deployment)
